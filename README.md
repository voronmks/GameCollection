🎮 Коллекция игр (список с оценками) – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольной коллекции игр на 7 языках программирования.
Приложение помогает вести учёт видеоигр: добавлять, редактировать, удалять, оценивать, фильтровать и получать статистику по вашей библиотеке.

🕹️ Что такое Коллекция игр?
Это личный каталог для видеоигр. Вы можете хранить информацию о каждой игре: название, жанр, платформу, год выпуска, статус (пройдена или нет), оценку (от 1 до 10), дату добавления и заметки. Приложение позволяет:

Добавлять игры с заполнением всех полей.

Просматривать весь список или фильтровать по жанру, платформе, статусу.

Искать по названию (частичное совпадение).

Сортировать список по оценке или названию.

Редактировать и удалять записи.

Получать статистику: общее количество, пройденные/непройденные, средняя оценка, распределение по платформам и жанрам.

Сохранять и загружать данные в формате JSON.

Экспорт и импорт в CSV (дополнительная фича в некоторых реализациях).

✨ Продвинутые возможности
Интерактивное меню с выбором действий.

Валидация ввода: оценка от 1 до 10, год от 1980 до текущего.

Автоматическая генерация ID и даты добавления.

Фильтрация по множеству критериев.

Сортировка по убыванию/возрастанию оценки или по алфавиту.

Статистика с разбивкой по платформам и жанрам.

Поддержка заметок (необязательное поле).

Сохранение в JSON и загрузка при старте.

Цветной вывод (где поддерживается) для улучшения читаемости.
